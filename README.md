## 🎭 The Irony

I built an AI therapist to help people with their mental health... and nearly lost mine in the process.

**Deployment attempts:**
- Render? ❌ *Out of memory (the server, not me... yet)*
- Railway? ❌ *Build timeout (like my patience)*

- Koyeb? ❌ *Wants credit card (I wanted my sanity)*
- Hugging Face Spaces? ✅ ***FINALLY!***

---

## 🌸 What is This?

Chitraksha is a mental wellness companion that uses RAG (Retrieval-Augmented Generation) to provide empathetic, context-aware support. Think of it as your AI friend who:
- Never judges your 3 AM existential crises
- Actually remembers what you talked about (unlike my goldfish)
- Has 2908 mental health resources in its brain
- Runs on free servers (because therapy shouldn't cost a kidney)

---

## ✨ Features That Somehow Work

### 🗣️ **AI Chat Companion**
- Powered by Groq's `llama-3.1-8b-instant` (fast enough to interrupt your overthinking)
- RAG engine with FAISS vector search (fancy words for "it actually knows stuff")
- Crisis detection with Indian helplines (because we care, dammit)
- Guest mode (for commitment-phobes)

### 📊 **Mood Tracker**
- Track your emotional rollercoaster
- Pretty graphs that make sadness look aesthetic
- Calendar view (proof that you survived this week)

### 🆘 **Resources**
- Breathing exercises (yes, you need to breathe)
- Grounding techniques (for when you're floating away)
- Indian mental health helplines
- Self-help articles (better than doom-scrolling)

### 🎨 **UI That Doesn't Hurt Your Eyes**
- Pastel colors (soothing like your grandma's living room)
- Lottie animations (because why not?)
- Dark mode (for late-night therapy sessions)
- Mobile-friendly (cry on the go!)

## 🏗️ Tech Stack (AKA My Suffering)

**Frontend:**
- Next.js 16 (bleeding edge, like my sanity)
- TypeScript (because JavaScript wasn't painful enough)
- Tailwind CSS (utility classes > my self-esteem)
- Framer Motion (smooth animations > my mental state)
- Deployed on Vercel (the only thing that worked first try)

**Backend:**
- Flask (simple, like my life goals)
- Groq API (fast, unlike this deployment process)
- FAISS (Facebook's gift to struggling devs)
- Sentence Transformers (turning words into numbers, turning me into a zombie)
- SQLite (lightweight, unlike my deployment struggles)
- Deployed on Hugging Face Spaces (free tier MVP)

## 📦 Installation (If You Dare)

### Prerequisites
- Node.js (and the will to live)
- Python 3.10+ (and coffee)
- A sense of humor (mandatory)

### Frontend Setup
```bash
cd frontend
npm install  # grab coffee
npm run dev  # pray
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt  # grab more coffee
python app.py  # pray harder
```
